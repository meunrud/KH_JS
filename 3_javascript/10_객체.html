Javascript 객체

        자바스크립트의 객체는 {} 내에 key:value (Map 형식) 가 모여있는 형태로 작성된다. 

        {k:v, k:v, k:v} //자바스크립트의 객체 형식

        [참고]
        자바스크립트 객체 모양의 문자열
        == JASON(Javascript Object Notation, 자바스크립트 객체 표기법)
        → "{k:v, k:v, k:v}"

        - 자바스크립트에 객체를 생성하는 방법
        1. {} :  객체 리터럴 표기법을 이용한 생성
        2. 생성자 + new 생성자()를 이용한 생성
